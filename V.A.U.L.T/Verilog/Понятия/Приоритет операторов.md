
| ~            | Отрицание                           |
| ------------ | ----------------------------------- |
| *,  /, %     | Умножение, деление, остаток         |
| +, -         | Сложение, вычитание                 |
| <<, >>       | Сдвиг влево/вправо                  |
| <<<, >>>     | Арифметический сдвиг влево/вправо   |
| <, <=, >, >= | Сравнение на больше/меньше          |
| ==, !=       | Сравнение на равенство/неравенсто   |
| &, ~&        | И, И-НЕ                             |
| ^, ~^        | Исключающее ИЛИ, исключающее ИЛИ-НЕ |
| \|, ~\|      | ИЛИ, ИЛИ-НЕ                         |
| ?:           | Условный оператор                   |
