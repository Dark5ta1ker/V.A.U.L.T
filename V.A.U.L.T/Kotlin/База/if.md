Оператор `if` 
```Kotlin
if (что-то сравнение с чем-то){
	делаем
} else if (дополнительное сравнение){
	делаем
} else {
	делаем
}
```
Можно сделать так:
```Kotlin
var strangeVar = if (что-то сравнение с чем-то){
	"делаем1"
} else if (дополнительное сравнение){
	"делаем2"
} else {
	"делаем3"
```
В `strangeVar` будет записана надпись, если сравнение дало `True`
